### Getting Started

``` sh
# install as a global package to make `broccoli-taco new` availble everywhere  
npm install -g broccoli-taco

# create a new site with some basic files and folders
broccoli-taco new my-site

# install dependencies
cd my-site && npm install

# run dev server
broccoli-taco serve

# Load index.html page
open http://localhost:4200/

# build complete site into dist/ folder
broccoli-taco build dist
```
